Imagine you have a special keyboard with the following keys:

Key 1: Prints A
 on screen

Key 2: (Ctrl−A)
: Select screen

Key 3: (Ctrl−C)
: Copy selection to buffer

Key 4: (Ctrl−V)
: Print buffer on screen appending it after what has already been printed.

Find the maximum numbers of A′s
 that can be produced by pressing keys on the special keyboard N
 times.

Input
The first line contains a single integer N
.(2≤N≤75).
Output
Your task is to return the maximum number of A′s
 that can be on the screen after performing N
 operations.

Examples
input
7
output
9
input
2
output
2
